**Fix:** Add the VoltExecutor folder to your antivirus exclusion list, then relaunch as Administrator.

### ❌ "Already Injected" Error Loops

**Cause:** A background process from a previous session is still running.
**Fix:**
```bash

```

### ❌ Mobile App Crashes on Launch

### ❌ Script Hub Won't Load
